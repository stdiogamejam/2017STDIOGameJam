        • Map dealCard()

          Removes a card from the card from the collection of cards. If there are no cards left, the deck is reshuffled before returning a card.

          Returns:
          a (Hash)Map of a card

        • ArrayList dealCards(int num)

          Deals a number of cards

          Returns:
          an arraylist holding the cards
          Parameters:

        • void initCards(int numDecks = 2)

          Using the numDecks variable, add the full set of cards to the cards collection that many times. The default value of numDecks is 2 so you can call this function with initCards() or initCards(). In Groovy, a JSON file is parsed into a collection of ArrayLists and Maps(dictionaries in Python). In an app with more space, you could wrap the representation of a card in a class.

          Parameters:
          numDecks

        • int rand(int max)

          Produces a random number under a certain value. Math.random generates a number between 0.0 and 1.0. Multiplying that by max turns it into a whole number.

          Returns:
          an integer
          Parameters:
          max

        • void shuffleDecks()

          Shuffles the deck of cards. Starting at the end of the cards, swap the positions of the selected card and one that appears in the collection of cards. Do this numDecks times.

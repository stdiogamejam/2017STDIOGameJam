Video Poker (in Groovy)

Released under MIT license

James Williams(James.L.Williams@gmail.com)

Measures 253 lines of code, according to CLOC.

Simulates single player video poker casino machines. Features in-game how to play instructions as well as
a payout chart that explains the cards required for each winning hand.

Uses JSON data files for the cards and payout structures.

To run as a script: Have groovy installed and execute `groovy videopoker.groovy`

You can also run with only Java installed by extracting the zip file.
Run `bin/videopoker`

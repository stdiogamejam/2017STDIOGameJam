In Finish the Lyrics you must finish the lyrics!

Installation:

1) You must have node installed https://nodejs.org/en/download/

2) Navigate to the folder in the command line

3) Run "node finish-the-lyrics"

Gameplay:

Players will be given a line from the song "Living on a Prayer" by Bon Jovi.
Players must type in the rest of the line.

Possible Extensions of the Game:
The lyrics.js file can be swapped out for lyrics from any song.
You could include multiple songs.
You could create a scraper to grab a random song from the internet.

Wanna Type
by Goat <goat@ridiculousglitch.com>

This is a classical typing game, using sentences from "Through the Looking
Glass" by Lewis Carroll. Players must type randomly chosen words (which appear
underlined) as fast as they can. After each turn, the sentences became longer.

- https://www.pythonanywhere.com/shared_console/32a67b30-65c8-4c24-b5ce-d3ab064ee318
- Files data.json and fonts.json are needed, run from the same directory
- Some values are hardcoded and should be put into data.json
